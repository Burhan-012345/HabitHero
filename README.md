# HabitHero - Habit Tracking & Social Platform

HabitHero is a comprehensive web application that combines habit tracking with social features, allowing users to build habits, share progress with friends, and stay motivated together.

### Core Features

- **Habit Tracking**: Create, manage, and track daily habits with streak counting
- **Social Snap Sharing**: Send disappearing photos/videos (snaps) to friends with 24-hour expiry
- **Real-time Chat**: Instant messaging with friends, including message status indicators
- **Friend System**: Add friends, send requests, and view each other's progress
- **Analytics Dashboard**: Visualize habit completion rates and progress over time
- **Profile Management**: Customizable profiles with avatars and bios

### Technical Features

- **Multi-account System**: Support for up to 10 accounts per email address
- **Email Verification**: Secure OTP-based email verification system
- **Real-time Updates**: WebSocket-powered notifications and status updates
- **File Upload**: Support for images and videos with proper storage management
- **Responsive Design**: Works on desktop and mobile devices
